Je réunis petit à petit des choses glanées ici où là, choisies pour leurs qualités particulières, de taille, de couleur, mais aussi de silence ou d'énigme. Commence ensuite le travail de composition sur le mur de l'atelier, qui se poursuit le temps de nouvelles trouvailles. Nombreux essais, des éléments sont enlevés, rajoutés, déplacés, transformés. Une impression de flux qui se serait figé comme la vie dans une photographie, est donnée par ces compositions. Une description minutieuse de tous les éléments composant chacun de ces 5 Murs les accompagne.
